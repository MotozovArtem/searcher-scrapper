7. Бригада: Мотозов Артем Владимирович, Шераухова Алина Александровна

Проект: Разработка приложения для сбора данных (Разработка скраппера)

Краткое описание: Приложение должно выполнять поиск сайтов, по ключевым словам, 
и собирать данные с N сайтов, где N – количество сайтов, которое необходимо обработать, 
задается пользователем. Ключевые слова задаются пользователем. 
Данные будут собираться в реляционную БД SQLite3.

Задачи:
* Провести анализ содержимого ответа от информационно поисковой системы
* Разработать схему базы данных
* Разработать модуль поиска данных
* Разработать модуль сбора данных 

Нет возражений. В итоге - приложение, работающее на отдельном компьютере (8 Gb ОП, 2 Gb видеопамять) 
без привлечения внешних ресурсов (серверов, баз данных и т.д.)
